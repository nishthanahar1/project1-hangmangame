# Hangman Game 🎯

A simple text-based Hangman game built using Python.

## Features
- Random word selection
- 6 attempts limit
- ASCII hangman animation
- Prevents repeated guesses

## Technologies Used
- Python
- Random module
- Loops and conditions

## How to Run
1. Open the project folder
2. Run:
   python hangman.py
